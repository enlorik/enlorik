## Projects

### User Management API

My first backend project at EPIC Institute.
*Spring Boot with proper layers — entities, repositories, services, controllers, auth.*

### pearl-proof

Math verification that actually checks the math.
*A C++17 + ANTLR4 checker that reduces both sides of an equation to canonical polynomial form and tests whether the difference is zero.*

### Bepo

A memory app. Eventually a Bangboo.
*Stores photos + notes + GPS, embeds them with CLIP so visual and text meaning share one search space, then uses cosine similarity for natural-language search.*

### HDD

A Codeforces dashboard, modeled on the HDD board from Zenless Zone Zero.
*Tracks upcoming contests and recommends daily problems by rating, filtered to unsolved. React/Vite + Express on Railway.*

### doc-flow

A backend shaped like real async work instead of plain CRUD.
*Spring Boot service for document jobs — queued, processed, and tracked by state, isolated per project.*

### grepink

A note system for AI answers I’d otherwise lose to chat history.
*Turns scattered conversations into notes I can search, trust, and reuse.*

### Ribbon

A debugging tool, because reading errors is slower than fixing them.
*Tauri app that clusters related errors and points at likely root causes.*
